## Open Source Contributions

### [vLLM Semantic Router](https://github.com/vllm-project/vllm)
**Python / vLLM / Distributed Inference / Mixture-of-Models**
Contributor to **vllm-sr**, a system-level intelligent router for mixture-of-models inference across cloud, data center, and edge. Landed a merged Python CLI PR and wrote a blog on semantic routing and agent gateways that drew strong community traction.

### [AgentGateway (CNCF)](https://github.com/AgentGateway/agentgateway)
**Rust / MCP / OAuth 2.1**
Added **Okta** as a first-class MCP auth provider with the full OAuth 2.1 flow (PR #1831). Implemented MCP resources support with multiplexing (PR #1896, resolving Issue #1858) operating in Rust on the agent/MCP data plane.

### [Perf to Firefox Profiler (Linux Kernel)](https://github.com/TwilightTechie)
**Linux Kernel / Python**
A converter bridging the Linux `perf` tool and the Firefox Profiler. Shipped directly in Linux kernel version 6.6 as part of my Google Summer of Code project with the Linux Foundation.

### [OpenClaw](https://github.com/openclaw)
**TypeScript / Ollama**
Fixed a cloud API-key discovery bug where `discoveryApiKey` failed to resolve for Ollama cloud endpoints, restoring cloud-model access in the agent runtime.
