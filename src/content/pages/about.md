Hello! My name is Anup and my background is systems all the way down. I'm an upstream Linux kernel contributor (perf tooling, shipped in kernel 6.6), a Google Summer of Code and Linux Kernel Mentorship Program alumnus, and before Nutanix I wrote WiFi 6E device drivers at MaxLinear used by millions to get online.

That low-level grounding — performance, debugging, concurrency, memory — is what I now bring to AI infrastructure. At Nutanix I work across distributed databases and AI-driven log analytics and anomaly detection, plus performance and debugging on distributed file systems in the core data path.

On the open-source side, I contribute to the infrastructure that makes LLMs run in production: vLLM Semantic Router for intelligent request routing, AgentGateway (CNCF) for MCP auth/multiplexing, and OpenClaw. I also run an always-on multi-LLM agent system at home on a Raspberry Pi cluster.
